World Module // Core Grid

City Block

The fundamental unit of the Blocktown map renderer. Each grid cell is exactly 260x260 pixels.

Instead of drawing independent roads and buildings, the engine assigns an inner 45px margin to represent half-width roads. The intersecting borders are dashed yellow lines, forming seamless street intersections when placed side-by-side.

The structure itself is constrained to a 170px footprint perfectly centered within this dark asphalt square.
